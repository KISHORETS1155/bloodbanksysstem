🩸 Blood Bank Availability System
📌 Description

The Blood Bank Availability System is a web-based application designed to help users quickly find available blood units during emergencies. It connects donors, hospitals, and patients through a centralized platform for searching blood availability, registering donors, and managing requests.

This system addresses real-world challenges like lack of awareness, slow search process, and fragmented data in blood management.

🌍 Community Impact

This project is developed to help:

🚑 Patients find blood quickly in emergencies
🏥 Hospitals manage blood inventory efficiently
❤️ Volunteers register as donors

It contributes to building a community-driven healthcare support system.

🚀 Features
🔍 Search blood by group and city
🧑‍🤝‍🧑 Donor registration system
🚨 Emergency blood request system
🏥 Blood bank inventory management
📊 Admin dashboard with analytics
📱 Fully responsive UI (mobile + desktop)
🏗️ Architecture

The system follows a 3-layer architecture:

UI Layer: HTML5, Bootstrap
Logic Layer: JavaScript modules
Storage Layer: LocalStorage (simulated database)

This modular design ensures scalability and maintainability.

🛠️ Tech Stack
HTML5
CSS3 & Bootstrap 5
JavaScript (ES6 Modules)
LocalStorage API
🔄 How It Works
User searches for blood by group and city
System filters available blood banks
Donors can register their details
Emergency requests can be submitted
Admin manages inventory and monitors activity
